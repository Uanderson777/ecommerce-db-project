# Desafio de Projeto: Sistema de E-commerce (Banco de Dados)

Este projeto foi desenvolvido como parte do desafio "Construindo seu Primeiro Projeto Lógico de Banco de Dados" da Digital Innovation One (DIO).

## 📌 Objetivo
Refinar um modelo de banco de dados para um cenário de E-commerce, aplicando conceitos de modelagem conceitual e lógica, além de implementar o esquema SQL.

## 🛠️ Implementações Realizadas
- **Especialização de Clientes:** Divisão entre Pessoa Física (PF) e Pessoa Jurídica (PJ) para garantir integridade.
- **Gestão de Pagamentos:** Registro de múltiplas formas de pagamento por cliente.
- **Logística de Entrega:** Controle de status de entrega e código de rastreio independente do pedido.
- **Integridade Referencial:** Uso de chaves primárias e estrangeiras para relacionar produtos, fornecedores e estoques.

## 📁 Arquivos do Repositório
- `schema.sql`: Script DDL com a criação das tabelas e definições de constraints.
- `queries.sql`: Consultas SQL para testes de persistência e recuperação de dados.
